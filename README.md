# 📗图书等级管理系统|C语言课设
## 文件说明
- `main.c`：主程序文件，包含程序入口和主要逻辑。
- `book.h`：头文件，包含数据结构定义和函数声明。
- `book.c`：实现文件，包含各个功能函数的具体实现。
- `files.h`: 文件操作相关函数的声明。
- `files.c`: 文件操作相关函数的实现。
- `SourceHanSansSC-Regular.otf`: 字体文件，用于程序中文显示。
- `chinese.txt`: 中文字符映射文件
- `icon.png`: 程序图标文件
- `book.png`: 书籍图标文件
- `book_library.data`: 存储图书信息的二进制数据文件。
- `raygui.h`：Raygui库头文件，用于图形界面开发。
